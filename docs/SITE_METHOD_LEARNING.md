# Site Method Learning Feature

## Overview
Автоматическое обучение предпочитаемых методов скрапинга для доменов. Система сама запоминает какой метод (HTTP или Chrome) лучше работает для конкретного сайта и использует его в первую очередь.

## Как это работает

### 1. Обучение
- Система автоматически записывает успехи/failures для каждого домена
- После 3+ успешных попыток метод становится "предпочитаемым"
- При будущих запросах к этому домену используется предпочитаемый метод первым

### 2. Забывание
- Если failure rate > 70%, предпочтение сбрасывается
- Данные старше 7 дней считаются устаревшими

### 3. Хранение
- Статистика сохраняется в YAML файл `./data/site_methods.yaml`
- При перезапуске сервера статистика загружается автоматически

## Конфигурация

```yaml
site_method:
  enabled: true                  # Включить обучение
  storage_dir: "./data"         # Директория для YAML файла
```

## Пример YAML файла

```yaml
domains:
  wowhead.com:
    domain: wowhead.com
    preferred_method: "HTTP"
    determined_at: "2026-06-03T17:31:33Z"
    success_count: 5
    failure_count: 1
    last_attempt: "2026-06-03T17:35:00Z"
```

## Логика выбора

1. **Если есть learned preference** → использовать его первым
2. **Если нет данных** → стандартная логика (JS → Chrome, иначе → HTTP)
3. **При success** → записать +1 к методу
4. **При failure** → записать +1, проверить failure rate
5. **Fallback** → если первый метод failed, попробовать второй

## Преимущества

✅ **Без хардкода** - автоматически работает для любого домена
✅ **Адаптивность** - изменяет предпочтения на основе опыта
✅ **Персистентность** - survives перезапуски сервера
✅ **Простота** - работает прозрачно, без вмешательства пользователя

## Когда полезно

- Сайты которые блокируют Chrome (Wowhead, LinkedIn, etc.)
- Медленные JS сайты где HTTP достаточно
- API endpoints где HTTP надежнее
- Любые домены с особыми требованиями

## Мониторинг

Можно проверить текущую статистику через логи:
```
INFO Loaded site method statistics domains_count=5
INFO Using learned method preference domain=wowhead.com learned_method=HTTP
INFO Recording first successful method for domain domain=reddit.com method=Chrome
```
